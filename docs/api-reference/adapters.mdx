# hud.adapters

The `hud.adapters` module provides adapter classes for converting between different action formats.

## Classes

### Adapter

The `Adapter` class is a base class for adapters that convert between different action formats.

#### Methods

##### convert

```python
def convert(self, data: Any) -> Dict[str, Any]:
    ...
```

Converts data from one format to another.

**Parameters:**
- `data` (Any): The data to convert

**Returns:**
- `Dict[str, Any]`: The converted data

##### adapt_list

```python
def adapt_list(self, actions: List[Any]) -> List[Dict[str, Any]]:
    ...
```

Converts a list of actions from one format to another.

**Parameters:**
- `actions` (List[Any]): The actions to convert

**Returns:**
- `List[Dict[str, Any]]`: The converted actions

### ClaudeAdapter

The `ClaudeAdapter` class is an adapter for converting Claude model outputs to HUD actions.

#### Methods

##### convert

```python
def convert(self, data: str) -> Dict[str, Any]:
    ...
```

Converts Claude output to HUD actions.

**Parameters:**
- `data` (str): Claude output text

**Returns:**
- `Dict[str, Any]`: The corresponding HUD action

## Common Action Types

### ClickAction

```python
ClickAction(x: int, y: int) -> Dict[str, Any]
```

Creates a click action at the specified coordinates.

**Parameters:**
- `x` (int): X coordinate
- `y` (int): Y coordinate

**Returns:**
- `Dict[str, Any]`: A click action

### TypeAction

```python
TypeAction(text: str) -> Dict[str, Any]
```

Creates a typing action with the specified text.

**Parameters:**
- `text` (str): Text to type

**Returns:**
- `Dict[str, Any]`: A type action

### ScrollAction

```python
ScrollAction(dx: int, dy: int) -> Dict[str, Any]
```

Creates a scroll action with the specified deltas.

**Parameters:**
- `dx` (int): Horizontal scroll amount
- `dy` (int): Vertical scroll amount

**Returns:**
- `Dict[str, Any]`: A scroll action

### DragAction

```python
DragAction(start: Point, end: Point) -> Dict[str, Any]
```

Creates a drag action from the start point to the end point.

**Parameters:**
- `start` (Point): Starting point
- `end` (Point): Ending point

**Returns:**
- `Dict[str, Any]`: A drag action

### Point

```python
Point(x: int, y: int) -> Dict[str, Any]
```

Creates a point with the specified coordinates.

**Parameters:**
- `x` (int): X coordinate
- `y` (int): Y coordinate

**Returns:**
- `Dict[str, Any]`: A point object
