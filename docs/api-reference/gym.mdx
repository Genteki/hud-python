The `hud.gym` module provides functions for creating and managing environments.

# Module Functions

## make

```python
make(env_id: str, job_id: str, **kwargs) -> Env
```

Creates a new environment instance.

**Parameters:**
- `env_id` (str): The ID of the environment to create
- `job_id` (str): The ID of the job to associate with the environment
- `**kwargs`: Additional keyword arguments to pass to the environment constructor

**Returns:**
- `Env`: A new environment instance
