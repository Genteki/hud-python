# hud.taskset

The `hud.taskset` module provides the TaskSet class for working with collections of tasks.

## Classes

### TaskSet

The `TaskSet` class represents a collection of related tasks. TaskSets are typically loaded using the `load` class method rather than being constructed directly.

#### Class Methods

##### load

```python
@classmethod
def load(cls, taskset_id: str) -> TaskSet:
    ...
```

Loads a TaskSet by its ID.

**Parameters:**
- `taskset_id` (str): The ID of the taskset to load

**Returns:**
- `TaskSet`: The loaded taskset

##### from_inspect_dataset

```python
@classmethod
def from_inspect_dataset(cls, dataset: Any) -> TaskSet:
    ...
```

Creates a TaskSet from an inspect-ai dataset.

**Parameters:**
- `dataset` (Any): An inspect-ai dataset

**Returns:**
- `TaskSet`: A new TaskSet instance
